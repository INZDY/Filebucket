# Obsidian-inspired vault browser layout

Filebucket uses an Obsidian-inspired web vault shape for MVP: a left Vault Browser for folders, nested folders, notes, and first-class media assets; a center Main Content Pane for Markdown editing and media preview; and a right Note Outline shown only for selected Markdown notes. This replaces the earlier folder side panel plus center item list model because the vault root and nested folders should browse like a file-oriented vault, notes remain Markdown-first, and media assets need direct browsing and preview without being hidden behind notes.
